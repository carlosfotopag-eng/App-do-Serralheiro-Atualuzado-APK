📖 MANUAL DO USUÁRIO - APP DO SERRALHEIRO PRO V2.0
🔨 SOBRE O APP
O APP DO SERRALHEIRO PRO V2.0 é uma ferramenta profissional desenvolvida para ajudar
serralheiros, marceneiros, telhadistas e profissionais da construção civil com cálculos precisos para
seus projetos.
📱 COMO USAR

2️⃣ CALCULADORA DE CIRCUNFERÊNCIA (📏 CIRC)
Para que serve: Calcular o comprimento de material necessário para fazer um círculo ou anel.
Como usar:
1. Digite o diâmetro (em mm) no campo indicado
2. Escolha se quer usar π real (3.1416) ou π aproximado (3.14)
3. O resultado aparece automaticamente
O que mostra:
·
📏 Comprimento do material em mm · 📐 Equivalente em metros
Dica útil: Use para calcular quanto material vai gastar para fazer flanges, anéis ou curvas circulares.3️⃣ MEDIÇÃO DE PEÇAS (📐 MEDIR)
Para que serve: Calcular ângulo, hipotenusa e comprimento total de peças inclinadas (escadas,
rampas, telhados, vigas inclinadas).
Como usar:
1. Ponto A (opcional) - Início da peça (mm)
2. Ponto B (obrigatório) - Distância horizontal / base (mm)
3. Ponto C (obrigatório) - Altura / desnível (mm)
O que o desenho mostra:
🔺

📐
·
Ponto A (amarelo) - início da peça ·
Ponto B (vermelho) - base horizontal ·
Ponto C
(verde) - altura que se move conforme o valor · Linha verde = hipotenusa (distância real inclinada)
Resultados calculados:
🔺
📐

➕
·
Ângulo de inclinação - inclinação da peça em graus ·
Hipotenusa - distância real do ponto
B ao ponto C ·
Comprimento total = Ponto A + Hipotenusa ·
Aumento linear - quanto a peça
aumenta devido à inclinação

4️⃣ CONVERSOR DE UNIDADES (🔄 CONV)
Para que serve: Converter medidas entre milímetros e polegadas.
Como usar:
· mm → polegadas: Digite o valor em mm, clique CONVERTER · in → mm: Digite o valor em
polegadas, clique CONVERTER
Dica útil: Muito útil quando você tem projetos em sistemas diferentes (métrica e imperial).5️⃣ NÍVEL DIGITAL (🎚️ NÍVEL)
Para que serve: Medir a inclinação do seu dispositivo em tempo real.
Como usar:
1. No iPhone: clique em "ATIVAR SENSOR" (primeira vez)
2. No Android: funciona automaticamente
3. Incline o celular para testar
O que mostra:
🎚️
📊
🟢
·
Ponteiro que se move como velocímetro ·
Ângulo exato em graus ·
Status colorido: ·
Verde: nivelado (0° ± 1.5°) · Amarelo: inclinação suave · Laranja: inclinação moderada · Vermelho:
inclinação acentuada
Exemplo prático:
· Verificar se uma superfície está nivelada · Medir inclinação de telhados e rampas
6️⃣ HISTÓRICO DE MEDIÇÕES
Para que serve: Armazenar as medições feitas na seção MEDIR (ABC).
Como usar:
1. Após calcular uma medição, clique em "SALVAR"
2. Clique em "HIST" para ver as medições salvas
3. Clique em "LIMPAR HISTÓRICO" para remover todas
Dica: As medições ficam salvas no navegador mesmo se você fechar o app.
📊 GRÁFICO COMPARATIVO
Após calcular na seção MEDIR, um gráfico de barras mostra a comparação visual entre:
📌 Ponto A · 📏 Base B · 📐 Altura C · 🔺 Hipotenusa · 🧱 Total

🔧 SUPORTE
Em caso de dúvidas sobre os cálculos:
· Ângulo: tan(θ) = altura / base · Hipotenusa: √(base² + altura²) · Circunferência: π × diâmetro
